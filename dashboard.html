Operational Dashboard

Building Dashboard using Power BI

Project Overview

Dataset : Microsoft Contoso BI Demo Dataset for Retail Industry.

Objective : The purpose of this project is to create an operational dashboard for sales performance tracking, which will enable the CEO of the company to keep track of the company's operations effectively. The dashboard will provide insights into sales performance, identify trends, and highlight key metrics and KPIs to aid decision-making processes.

Implement

1. Query data using SQL Sever

2. Creating Dashboard

Dashboard is separated into 3 parts: Overview, Product, and Store.

a. Overview: Tracking sales trends

b. Product: Order and Revenue by Product Brand, Top 5 Products by Revenue and Order

c. Store: Compare Revenue, Profit, and Order by Store, Country, and City
